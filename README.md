# rp42.at

Persönliche Visitenkarte von Robert Preinfalk – ein minimalistisches, statisches One-Pager-Landingpage-Design.

## Inhalt

- `index.html` – Seitenstruktur (Portrait, Name/Titel, Kontaktlinks, QR-Code, Kontaktformular)
- `styles.css` – Styling (Typografie, Farbpalette, Fade-in-Animationen)
- `assets/rp42.jpg` – Porträtfoto
- `assets/qr-code.png` – QR-Code mit vCard-Kontaktdaten

Keine Build-Schritte nötig – die Seite ist rein statisches HTML/CSS und kann direkt gehostet werden.
